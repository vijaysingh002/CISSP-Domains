The CISSP (Certified Information Systems Security Professional) certification consists of 8 domains:

  1. Security and Risk Management: Focuses on the principles of security, risk, and compliance.
  2. Asset Security: Protecting data and assets to ensure confidentiality, integrity, and availability.
  3. Communication and Network Security: Securing network architecture and ensuring safe data transfer.
  4. Security Architecture and Engineering: Designing and building secure systems.
  5. Identity and Access Management (IAM): Controlling user access and protecting user identities.
  6. Security Operations: Managing and monitoring security operations and incident responses.
  7. Security Assessment and Testing: Evaluating and testing the effectiveness of security measures.
  8. Software Development Security: Securing the software development process.
